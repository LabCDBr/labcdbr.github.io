_n()

i.php : 255

Retrieve the plural or single form based on the supplied amount.

public static _n(string $single, string $plural, int $number[, string $domain = 'default' ]) : string

If the text domain is not set in the $l10n list, then a comparison will be made and either $plural or $single parameters returned.

If the text domain does exist, then the parameters $single, $plural, and $number will first be passed to the text domain's ngettext method. Then it will be passed to the 'ngettext' filter hook along with the same parameters. The expected type will be a string.

Parameters
$single : string

the text that will be used if $number is 1

$plural : string

the text that will be used if $number is not 1

$number : int

the number to compare against to use either $single or $plural

$domain : string = 'default'

Optional. Text domain. Unique identifier for retrieving translated strings.

Return values
string —

either $single or $plural translated text

_n_noop()

i.php : 306

Register plural strings in POT file, but don't translate them.

public static _n_noop(string $singular, string $plural[, string $domain = null ]) : array<string|int, mixed>

Used when you want to keep structures with translatable plural strings and use them later.

Example:

$messages = array(
 	'post' => _n_noop( '%s post', '%s posts' ),
 	'page' => _n_noop( '%s pages', '%s pages' ),
);
...
$message = $messages[ $type ];
$usable_text = sprintf( translate_nooped_plural( $message, $count ), $count );
Parameters
$singular : string

single form to be i18ned

$plural : string

plural form to be i18ned

$domain : string = null

Optional. Text domain. Unique identifier for retrieving translated strings.

Return values
array<string|int, mixed> —

array($singular, $plural)

replaces()

i.php : 474

Substitui termos das strings traduzidas pelo o que estiver definido no arquivo translations/replacements.

public static replaces(mixed $translation, mixed $domain) : mixed

o arquivo translations/replacements deve seguir a seguinte estrutura:

espao >>> museu minhas oportunidades >>> meus editais suas oportunidades >>> seus editais as oportunidades >>> os editais uma oportunidade >>> um edital a oportunidade >>> o edital oportunidades >>> editais oportunidade >>> edital espaos culturais >>> espaos de sade

os termos sero processados da seguinte maneira: para a linha abaixo:

meus espaos >>> meus museus

sero aplicadas as seguintes funes para substituio:

ucwords => "Meus Espaos >>> Meus Museus" ucfirst => "Meus espaos >>> Meus museus" mb_strtolower => "MEUS ESPAOS >>> MEUS MUSEUS" mb_strtolower => "meus espaos >>> meus museus"
